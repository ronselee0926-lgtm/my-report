# AI报告生成提示词指引

## 📋 快速开始

在AI输入框中直接输入你的报告内容，AI会自动提取数据并生成图表。

---

## 🎯 提示词基本结构

一个完整的提示词应包含以下要素：

```
1. 报告标题
2. 章节结构（使用一、二、三或1. 2. 3.）
3. 具体数据（数字、百分比、金额等）
4. 图表要求（可选，AI会自动判断）
```

## 📊 支持的图表类型

### 1. 柱状图 (chart)
**适用场景**：比较不同类别的数值

**提示词写法**：
```
各部门销售额：
销售部 500万
技术部 300万
市场部 200万
```

### 2. 折线图 (line)
**适用场景**：展示趋势变化

**提示词写法**：
```
1月销售额100万，2月120万，3月150万，4月130万
```

### 3. 饼图 (pie)
**适用场景**：展示占比分布

**提示词写法**：
```
市场份额：A产品40%，B产品30%，C产品20%，其他10%
```

### 4. KPI指标卡 (kpi)
**适用场景**：展示关键指标

**提示词写法**：
```
核心指标：
- 用户数 10万（+15%）
- 营收 500万（+25%）
- 转化率 3.5%（+0.5%）
```

### 5. 进度条 (progress)
**适用场景**：展示完成进度

**提示词写法**：
```
项目进度：
- 项目A 75/100
- 项目B 60/100
- 项目C 90/100
```

### 6. 表格 (table)
**适用场景**：展示详细数据

**提示词写法**：
```
| 月份 | 销售额 | 成本 | 利润 |
|------|--------|------|------|
| 1月  | 100万  | 60万 | 40万 |
| 2月  | 120万  | 70万 | 50万 |
```

### 7. 瀑布图 (waterfall)
**适用场景**：展示累计变化

**提示词写法**：
```
利润分析：
期初利润 100万
+ 收入增长 50万
- 成本增加 -30万
= 期末利润 120万
```

### 8. 漏斗图 (funnel)
**适用场景**：展示转化流程

**提示词写法**：
```
转化漏斗：
访问 10000人
注册 3000人
下单 800人
支付 500人
```

### 9. 雷达图 (radar)
**适用场景**：多维度对比

**提示词写法**：
```
产品能力评估：
产品A：速度80分，质量90分，价格70分，服务85分
产品B：速度70分，质量85分，价格90分，服务80分
```

### 10. 热力图 (heatmap)
**适用场景**：展示矩阵数据

**提示词写法**：
```
用户活跃度（时段 vs 星期）：
周一：早10%，中20%，晚30%
周二：早15%，中25%，晚35%
周三：早20%，中30%，晚40%
```

### 11. 甘特图 (gantt)
**适用场景**：项目时间管理

**提示词写法**：
```
项目时间表：
任务A：2024-01-01至2024-01-10，进度80%
任务B：2024-01-05至2024-01-15，进度60%
任务C：2024-01-12至2024-01-20，进度30%
```

### 12. 词云 (wordcloud)
**适用场景**：展示关键词频率

**提示词写法**：
```
用户反馈关键词：
质量好（100次）、物流快（80次）、价格贵（70次）、
客服好（60次）、包装差（40次）
```

## 💡 提示词技巧

### 1. 数据格式
- **数字**：直接写数字，如 `500万`、`12,500`、`25%`
- **时间**：使用标准格式，如 `2024-01-01`、`1月`、`Q1`
- **对比**：使用"环比"、"同比"、"增长"、"下降"等关键词

### 2. 结构清晰
- 使用标题层级（一、二、三或1. 2. 3.）
- 使用列表符号（- 或 *）
- 使用表格（Markdown表格格式）

### 3. 明确图表要求
如果需要特定图表，可以在最后说明：
```
请生成：
- 柱状图展示销售额对比
- 折线图展示趋势变化
- 饼图展示占比分布
```

### 4. 混合使用
可以在同一份报告中混合多种数据格式：
```
一、总体情况（文字描述）
二、详细数据（表格）
三、趋势分析（折线图数据）
四、占比分布（饼图数据）
```

---

## ⚠️ 注意事项

1. **数据准确性**：确保输入的数字准确，AI会直接使用这些数字生成图表
2. **单位统一**：同一指标保持单位一致（如都用"万"或都用"元"）
3. **避免歧义**：清晰标注每个数字的含义
4. **适量数据**：单个图表建议不超过10个数据点，过多会影响可读性

## 🔍 常见问题

### Q1: AI没有生成我想要的图表类型？
**A**: 在提示词末尾明确指定："请生成柱状图/饼图/折线图..."

### Q2: 图表数据不对？
**A**: 检查输入的数字是否清晰可识别，避免"约"、"左右"等模糊词汇

### Q3: 生成的文字内容太少？
**A**: 在提示词中多写一些描述性文字，AI会将其转换为text组件

### Q4: 如何生成多个图表？
**A**: 在提示词中提供多组数据，AI会自动为每组数据生成相应图表

## 📌 最佳实践

1. **先写框架**：先列出报告的大纲结构
2. **填入数据**：在每个章节下填入具体数据
3. **补充说明**：添加背景说明和分析结论
4. **指定图表**：明确需要的图表类型（可选）
